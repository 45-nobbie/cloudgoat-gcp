# Portal (scaffold)

This folder contains the React + Node scaffold for the portal UI and backend.

## Local dev
1. `cd portal/scaffold`
2. `npm install`
3. `npm run dev`

## What to implement next
- React pages: Landing, Dashboard, Challenge, Leaderboard, Admin.
- Backend: simple Node API for flag validation, using Secret Manager in production.
- Styling: Tailwind CSS and component library (shadcn/ui recommended).
